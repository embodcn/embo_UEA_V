## Contenido
Este repositorio contiene las asignaturas dadas de la UEA V: Comunicación creativa en la cultura digital. Se organizarán de acuerdo al tiempo determinado. Poco a poco se irán incorporando temas de interés y se irán añadiendo materiales personales que pueden ser de utilidad para toda la comunidad. 
## Reflexión de mi relación con las tecnologías web
Mi relación con las tecnologías web (HTML, CSS, JS) se podrían definir de forma ambivalente. Por un lado me han resultado relativamente complejas, porque nunca antes tuve la oportunidad de aprender programación para la web, sin embargo, acepto que también nunca tuve el interés por acercarme a estos temas. Por otro lado, y en base a lo anterior, me ha resultado bastante interesante poder comprender esta complejidad con una mentalidad abierta y fresca. Anteriormente, al navegar en páginas web, no comprendía las etiquetas de complejidad y estructura lógica que tienen detrás en cada una de ellas, pero al empezar a crear mis propios sitios y ver el código estructurado, ahora comprendo que hay toda una arquitectura ideológica y lógica que ayudan a que cobren vida estos sitios web.
